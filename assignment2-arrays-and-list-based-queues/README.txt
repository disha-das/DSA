This executable implements Queues as a Data Structure in 3 different modes:
mode1: regular array
mode2: circular array index counted circularly with empty cell
mode3: circular list

It also provides the Josephus survivor solution in mode 1 queue.

Three queues can be maintained simultaneously.
Mode and operation are taken as input for before the operation is performed.
The operations inlude:
Enqueue
Dequeue
PrintQueue
(Available for every mode)
